📍 GeoÁrea GNSS

Para utilizar o rastreamento GNSS, permita o acesso à localização quando solicitado.

Importante: Este aplicativo precisa ser aberto em HTTPS (ou localhost) para acessar a geolocalização. Se estiver em rede local, use um servidor HTTPS simples.

Você também pode tocar no mapa para adicionar pontos manualmente.

⚠️ Permissão de localização negada

O aplicativo não pode acessar os sinais GNSS porque a permissão foi bloqueada. Para resolver:

  • 🔓 No seu dispositivo, vá em Configurações > Apps > Navegador > Permissões > Localização e escolha "Permitir".
  • 🌐 Certifique-se de que o site está sendo acessado via HTTPS (cadeado verde na barra de endereço).
  • 🔄 Após alterar as configurações, clique no botão abaixo para tentar novamente.
